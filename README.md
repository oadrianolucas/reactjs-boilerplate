First, run the development server:

```bash

# or
yarn dev
```

## Commands

- `dev`: application on `localhost:3000`
- `build`
- `start`
- `lint`

- `test:watch`
- `postinstall`
- `prepublishOnly`
- `postpublish`
- `prepare`
